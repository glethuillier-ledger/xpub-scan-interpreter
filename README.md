Xpub Scan reports interpeter.

This tool suggest interpretations of Xpub Scan reports in human language.

## Overview

![Flow](./flow.png)

## Install

```
$ yarn
$ yarn build
```

## Use

```
$ node lib/main.js {path to Xpub Scan JSON report}
```

Example:

```
$ node lib/main.js ./abcd.json
```
